# 16. 迁移自定义元素与 is 用法

## 当前任务

修正自定义元素识别和 `is` 属性的变化。

## 怎么做

1. 搜索 `ignoredElements`、`is="`。
2. 如果项目有自定义元素白名单，把 `ignoredElements` 改到编译器配置里的 `isCustomElement`。
3. 如果原来在普通原生标签上用 `is` 渲染 Vue 组件，改成 `<component :is="...">`。
4. 如果是 in-DOM template 的特殊场景，按文档要求使用 `vue:` 前缀。

## 改完以后确认

- 自定义元素不再被错误当成 Vue 组件。
