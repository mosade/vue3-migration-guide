# 09. 迁移函数式组件

## 当前任务

去掉 Vue 2 的函数式组件声明方式。

## 怎么做

1. 搜索 `<template functional>` 和 `functional: true`。
2. 对 `<template functional>` 组件，去掉 `functional`，并把原来的 `props`、`attrs`、`listeners` 使用方式改成 Vue 3 对应写法。
3. 对 `functional: true` 组件，改成普通函数组件。

## 改完以后确认

- 项目里不再出现 `<template functional>` 和 `functional: true`。
