# 10. 迁移异步组件写法

## 当前任务

把 Vue 2 的异步组件写法改成 Vue 3 的 `defineAsyncComponent`。

## 怎么做

1. 搜索 `() => import(`、`component: (`、`resolve, reject` 这种旧异步组件工厂写法。
2. 统一改成 `defineAsyncComponent(() => import(...))`。
3. 如果用了异步组件配置对象，把 `component` 改成 `loader`，`loading` 改成 `loadingComponent`，`error` 改成 `errorComponent`。

## 改完以后确认

- 所有异步组件都已经通过 `defineAsyncComponent` 创建。
