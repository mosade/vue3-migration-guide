# 15.移除 .sync

- 目标：移除旧版双向绑定语法依赖。
- 搜索线索：全局搜索 `.sync`、`update:`、包装组件上的透传属性。
- 实施内容：优先将 `:foo.sync="bar"` 改为 `v-model:foo="bar"`；如果组件不是标准表单组件，也可以保留显式 `:foo="bar" @update:foo="bar = $event"`。
- 注意事项：不要只改父组件模板，子组件也必须确认是否真的触发 `update:foo`；多个 `.sync` 在 Vue 3 下通常对应多个带参数的 `v-model`。
- 完成标准：父子同步链路在编辑、重置、异步回填三种场景下都保持可用。
