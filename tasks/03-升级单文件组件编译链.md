# 03.升级单文件组件编译链

- 目标：让 `.vue` 文件编译链兼容 Vue 3。
- 搜索线索：检查 `vue-loader`、`@vue/cli-service`、webpack 配置、Babel/TS 配置、测试编译配置。
- 实施内容：升级或调整 SFC loader/plugin，使 `<template>`、`<script>`、`<style scoped>`、预处理器都按 Vue 3 规则处理。
- 注意事项：大型老项目常有手写 webpack rule，必须确认没有重复处理 `.vue` 或把 Vue 2 loader 留在分支配置里。
- 完成标准：普通 SFC、带 scoped style 的 SFC、带预处理器的 SFC 都能编译通过。
