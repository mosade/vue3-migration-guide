# 01.升级 vue 到 3.x

- 目标：将 `vue` 主依赖升级到 Vue 3 正式版本。
- 搜索线索：检查 `package.json` 中 `vue`、`vue-template-compiler`、`vue-loader`、`vue-router`、`vuex`、UI 组件库及 Vue 插件。
- 实施内容：直接升级到 Vue 3，不引入 `@vue/compat`；同步清理与 Vue 2 强绑定的 peer/version 约束。
- 注意事项：依赖安装成功不代表迁移完成，后续任务会继续处理编译链、入口、router、store 和运行时代码。
- 完成标准：依赖树中主运行时为 Vue 3，且没有继续安装 Vue 2 专用编译器作为有效依赖。
