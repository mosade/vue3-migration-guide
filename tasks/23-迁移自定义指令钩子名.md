# 23.迁移自定义指令钩子名

- 目标：迁移自定义指令生命周期名称。
- 搜索线索：全局搜索 `Vue.directive`、`directives:`、`bind(`、`inserted(`、`componentUpdated(`、`unbind(`。
- 实施内容：`bind` 改为 `beforeMount`，`inserted` 改为 `mounted`，`componentUpdated` 改为 `updated`，`unbind` 改为 `unmounted`。
- 注意事项：如果指令依赖组件实例，Vue 3 的 `binding.instance` 行为要单独确认；如果指令操作 DOM 或注册全局事件，必须检查清理时机。
- 完成标准：指令初始化、更新、卸载都在预期时机执行，且没有遗留事件监听或定时器。
